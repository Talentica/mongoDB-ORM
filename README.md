# mongoDB-ORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm start

# development - watch mode
$ npm run start:dev

# lint
$ npm run lint

# build
$ npm run build

# test
$ npm run test

# test - watch mode
$ npm run test:watch

# coverage
$ npm run test:cov
```
